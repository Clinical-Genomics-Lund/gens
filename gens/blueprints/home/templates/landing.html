GENS - v{{ version }}

Gens is a web-based interactive tool to visualize genomic copy number profiles from WGS data. It plots the normalized read depth and alternative allele frequency. Gens currently does not attempt to visualize breakpoint information, so use Scout/IGV for that! The way we generate the data it is suitable for visualizing CNVs of sizes down to a couple of Kbp, for smaller things use Scout/IGV!
